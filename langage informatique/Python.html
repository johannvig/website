Python



Python est un langage de programmation de haut niveau à usage général et largement utilisé. Il a été initialement conçu par Guido van Rossum en 1991 et développé par Python Software Foundation. Il a été principalement développé pour mettre l’accent sur la lisibilité du code, et sa syntaxe permet aux programmeurs d’exprimer des concepts en moins de lignes de code.

En 1989, il travaille sur un système distribué appelé Amoeba. Guido développait des utilitaires pour le système quand il a réalisé que le langage C ralentissait le développement. Il a donc construit un nouveau langage qui lui permettra de travailler plus rapidement. Ce langage devait se situer entre le C et un script shell qui était plus difficile à lire. C’est donc de là qu’est né Python, nommé d’après une troupe de comédie surréaliste britannique Monty Python.

Le premier ordinateur personnel avec une interface graphique, ou interface utilisateur graphique - une interface qui permet aux utilisateurs d'interagir avec des icônes visuelles - était l'Apple Lisa. La toute première interface graphique a été développée par Xerox Corporation dans son centre de recherche de Palo Alto (PARC) dans les années 1970. Steve Jobs a visité le PARC en 1979 (après avoir acheté le stock Xerox) et a été impressionné et fortement influencé par le Xerox Alto, le premier ordinateur doté d'une interface graphique. Cette machine, cependant, était assez grande. Jobs a adapté la technologie pour l'Apple Lisa, un ordinateur suffisamment petit pour tenir sur un bureau.

Le langage est finalement sorti en 1991. Lors de sa sortie, il utilisait beaucoup moins de codes pour exprimer les concepts, lorsque nous le comparons avec Java, C ++ & C. Sa philosophie de conception était également assez bonne. Son objectif principal est de fournir une lisibilité du code et une productivité avancée des développeurs. Quand il a été publié, il avait une capacité plus que suffisante pour fournir des classes avec héritage, plusieurs types de données de base, gestion des exceptions et fonctions.

Vers 2010, le langage a commencé à rivaliser avec les meilleurs langages de programmation tels que Java et JavaScript. La croissance des domaines tels que le machine learning, le big data et particulièrement la sortie de TensorFlow a fait augmenter l’intérêt mondial pour Python.
